# GitPub API

This is a simple REST API for managing quizzes. Each quiz consists of:
- Quiz name
- List of questions, where each question contains:
  - Question text
  - Correct answer text

## Features
The API supports the following operations:

- Fetch all quizzes
- Fetch all questions
- Create a new quiz
- Edit an existing quiz
- Delete a quiz
- Export a quiz
